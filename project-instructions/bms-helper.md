# BMS-Helper Project Instructions

## Issue Creation

When creating issues related to the `bms-helper` project, create them in the **bms-full** project instead:

- **Project URL:** `candu/bms-full`
- **Issue Tracker:** http://services.conexusnuclear.org:8929/candu/bms-full/-/issues

This ensures all BMS-related issues are tracked in a single location.

# Related Project
This project is the API portion of the BMS. The "bms" project is the frontend associated with this API project.
